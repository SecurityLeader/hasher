# hasher

Hasher is a Hash Cracker that has supported more than 7 types of hashes

# hasher feature

✅ auto detect hash

✅ supports windows and linux platforms

✅ fast decrypt

✅ list of supported hashes:

- md4
- md5
- sha1
- sha224
- sha256 
- sha384
- sha512
- ripemd160
- whirlpool
- mysql 3.2.3
- mysql 4.1
- mssql2000
- mssql2005
- nthash
- lmhash
- ntlm hash

#[ Installation ]

$ apt update upgrade

$ python2 hash.py
